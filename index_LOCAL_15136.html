Git - gitinho

Trabalhando em Equipe com Controle e Segurança

Um curso, da alura, que explora os benefícios de utilizar o Git como ferramenta de controle de versão para projetos em qualquer linguagem, em qualquer plataforma.

Este curso é um prerequisito da formação "Engeenheiro Front-End"

Principais benefícios:

  • Funciona de maneira distribuída
  • Permite a edição concorrente de arquivos do projeto
  • Não depende de uma conexão ativa com um servidor
  • Possui ferramentas de gestão de conflitos
  • etc.
